## ✨ **Features**
- ✅ **Unlimited trades** - Add as many as you want
- 📸 **2 Image uploads** - Before & After trade screenshots
- 💱 **Any Symbol** - XAU/USD, EUR/USD, BTC/USD, etc.
- 📊 **Live Stats** - Total trades + Win Rate %
- 💾 **Auto-save** - Data stored in your browser
- 📱 **Mobile-first** - Perfect on phone/tablet
- 🗑️ **Delete trades** - Individual or Clear All
- ⚡ **No server needed** - Pure HTML/CSS/JS
